# Django-сайт для классификации отходов

Веб-приложение для классификации отходов с использованием методов машинного обучения.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Примените миграции:
```bash
python manage.py migrate
```

3. Создайте суперпользователя (опционально):
```bash
python manage.py createsuperuser
```

4. Запустите сервер:
```bash
python manage.py runserver
```

5. Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/

## Структура проекта

- `waste_classification/` - основной проект Django
- `classifier/` - приложение для классификации
  - `ml_model.py` - модель машинного обучения
  - `views.py` - представления (views)
  - `templates/` - HTML шаблоны
- `static/` - статические файлы (CSS, JS)
- `media/` - загружаемые пользователями изображения

## Функционал

1. **Главная страница** - информация об учебном пособии
2. **Практика машинного обучения** - загрузка и классификация изображений отходов
3. **Теория машинного обучения** - раздел в разработке

## Модель

Модель использует архитектуру CNN (сверточная нейронная сеть) для классификации отходов на 6 категорий:
- Glass (Стекло)
- Metal (Металл)
- Organic (Органические отходы)
- Paper (Бумага)
- Plastic (Пластик)
- Trash (Прочий мусор)

## Примечание

Для улучшения точности классификации рекомендуется обучить модель на вашем датасете и сохранить веса в файл `classifier/model_weights.h5`.
